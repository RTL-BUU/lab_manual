Bambu Lab A1 Printing Guide (คู่มือการพิมพ์ 3 มิติ)

เครื่องพิมพ์ 3 มิติรุ่น Bambu Lab A1 เป็นเครื่องพิมพ์ระบบ FDM ความเร็วสูง ที่แลปของเราเลือกใช้ ในคู่มือนี้จะเน้นขั้นตอนการใช้งานสำหรับผู้ที่มีไฟล์โมเดลพร้อมพิมพ์แล้ว

1. Software Setup (เตรียมซอฟต์แวร์)

เราจะไม่ใช้โปรแกรม Slicer ทั่วไป (เช่น Cura) แต่จะใช้โปรแกรมเฉพาะของค่ายนี้เพื่อประสิทธิภาพสูงสุด

  • Download: โหลดโปรแกรม Bambu Studio ได้ที่เว็บไซต์ทางการของ Bambu Lab (รองรับ Windows/Mac)

  • Installation: ติดตั้งและ Log in ด้วยบัญชีของแลป (หรือบัญชีส่วนตัวถ้าเชื่อมต่อผ่าน Cloud Mode)

2. File Formats (ประเภทไฟล์งาน)

เมื่อออกแบบชิ้นงานเสร็จแล้ว (เช่นจาก SolidWorks) เราต้องเลือกนามสกุลไฟล์ให้ถูกต้อง

  • STL (.stl): รูปแบบมาตรฐานดั้งเดิม เก็บเฉพาะผิวชิ้นงาน (Mesh) ไม่มีการระบุสีหรือหน่วยวัดที่แน่นอน (อาจเจอปัญหาขนาดเพี้ยนถ้าตั้งค่าหน่วยผิด)

  • OBJ (.obj): คล้าย STL แต่เก็บรายละเอียดได้มากกว่า นิยมในงานปั้นโมเดล

  • 3MF (.3mf): (แนะนำ) รูปแบบสมัยใหม่ เก็บข้อมูลได้ครบถ้วนทั้งขนาด สี วัสดุ และการตั้งค่าการพิมพ์ ไฟล์มีขนาดเล็กกว่า STL

  • G-Code (.gcode): คือไฟล์คำสั่งที่เครื่องพิมพ์อ่านรู้เรื่อง (ภาษาเครื่อง) เราไม่ได้ Export เป็น G-Code โดยตรง แต่ Slicer จะแปลงไฟล์ 3D ให้เป็น G-Code

สรุป: ปัจจุบันแลปยังใช้ STL เป็นหลัก แต่หากต้องการความแม่นยำสูงสุดแนะนำให้ลองเปลี่ยนมาใช้ 3MF

3. Workflow: From CAD to Print

ขั้นตอนการทำงานสำหรับผู้ใช้งาน SolidWorks มีดังนี้:

  1. SolidWorks: ไปที่ File > Save As เลือกประเภทไฟล์เป็น STL

    • Tip: กดปุ่ม Options ก่อน Save แล้วปรับค่า Deviation และ Angle ให้ต่ำลง เพื่อให้ผิวชิ้นงานโค้งมนไม่เป็นเหลี่ยม

  2. Bambu Studio: เปิดโปรแกรม แล้วลากไฟล์ STL เข้ามาวาง

  3. Prepare: จัดวางชิ้นงานบนฐาน (Plate) หมุนให้ด้านที่เรียบที่สุดอยู่ด้านล่างเพื่อลดการใช้ Support

4. Material & Profile Settings (การตั้งค่าวัสดุ)

ที่แลปใช้เส้นพลาสติกยี่ห้อ eSun ชนิด PLA+ เป็นหลัก เนื่องจากมีความเหนียวมากกว่า PLA ปกติ

วิธีโหลด Profile (สำคัญ)

  1. ใน Bambu Studio ช่อง Filament

  2. ไม่ต้องใช้ Generic PLA ให้กดรูปเฟือง หรือเลือกเมนู Download Filament System Preset

  3. ค้นหาและติ๊กถูกที่ “eSun PLA+”

  4. โปรแกรมจะโหลดค่าอุณหภูมิและความเร็วที่จูนมาแล้วสำหรับเส้นนี้โดยเฉพาะ

ความแตกต่างของวัสดุ

  • PLA/PLA+: พิมพ์ง่าย ไม่เหม็น แข็งแรงปานกลาง ไม่ทนร้อน (เกิน 60°C จะเริ่มนิ่ม) -> ใช้เป็นหลักในแลป

  • PETG: เหนียว ทนร้อนได้ดีกว่า PLA พิมพ์ยากกว่าเล็กน้อย

  • ABS: ทนร้อนสูง ขัดแต่งง่าย แต่พิมพ์ยากมาก หดตัวสูง และมีกลิ่นเหม็น (ไม่แนะนำให้ใช้กับ A1 ซึ่งเป็นเครื่องเปิด Open Frame)

  • TPU: พลาสติกนิ่ม (ยาง) ยืดหยุ่นได้ พิมพ์ยาก ต้องใช้ความเร็วต่ำ

5. Slicer Settings (Intermediate)

ค่าที่ต้องตรวจสอบและปรับเปลี่ยนตามความเหมาะสมของชิ้นงาน

  • Strength (ความแข็งแรง):

    • Wall Loops: แนะนำ 3 รอบ (แข็งแรงกว่าเพิ่ม Infill)

    • Infill Density: ปกติ 15-20% ก็เพียงพอ หากรับน้ำหนักมากใช้ 40-50%

    • Infill Pattern: แนะนำ Gyroid (แข็งแรงทุกทิศทาง หัวพิมพ์ไม่กระแทก) หรือ Grid (พิมพ์เร็วแต่หัวพิมพ์อาจชน)

  • Support (ตัวค้ำยัน):

    • Type: แนะนำ Tree (Auto) แกะง่าย ประหยัดพลาสติก และผิวชิ้นงานสวยกว่าแบบ Normal

    • Overhang Angle: มุมที่เริ่มสร้าง Support ค่ามาตรฐานคือ 30-45 องศา

  • Bed Adhesion (การยึดเกาะฐาน):

    • Skirt: พิมพ์เส้นรอบๆ เช็คระดับหัวพิมพ์ (ไม่ได้ช่วยยึดเกาะ)

    • Brim: พิมพ์ขยายฐานออกมาติดกับชิ้นงาน ช่วยแก้ปัญหาชิ้นงานร่อน (Warping) -> แนะนำให้เปิดถ้าฐานชิ้นงานเล็ก

    • Raft: พิมพ์แพรองรับทั้งชิ้นงาน (เปลืองพลาสติก ไม่ค่อยจำเป็นสำหรับ A1)

6. Sending to Printer (สั่งพิมพ์)

เครื่อง Bambu Lab A1 ที่แลปเชื่อมต่อ WiFi ชื่อ RTL ไว้แล้ว

  1. กดปุ่ม Slice Plate

  2. ตรวจสอบภาพจำลอง (Preview) ดูว่า Support ขึ้นครบถ้วนหรือไม่

  3. กดปุ่ม Print Plate (มุมขวาบน)

  4. เลือกชื่อเครื่องพิมพ์ (Printer) ให้ถูกต้อง

  5. กด Send (ไฟล์จะถูกอัปโหลดผ่าน Cloud/LAN ไปที่เครื่องและเริ่มพิมพ์ทันที ไม่ต้องเสียบ SD Card)

Tip

Pro Tip: หาก WiFi มีปัญหา สามารถเลือก Export G-code file ลง SD Card แล้วนำไปเสียบที่หน้าเครื่องได้เช่นกัน
